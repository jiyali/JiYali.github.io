页面底部跳动的爱心

效果图：

实现：

首先找到图标库，如我现在选择下图中的爱心，那么我要复制的内容就是下图红框中的 fa-heartbeat。
打开博客主目录文件 -> themes -> next -> _config.yml -> 搜索 footer -> 将 name 后面的 user 替换成 fa-heartbeat -> 将 animated 设置成 true -> 将 color 修改为 #ff0000
